### Это уже код с весами, если есть картинка и нужно проверить, то введите команду

```bash

python main.py --predict test.jpg

```

### Если задача стоит, самому проверить и потренировать модель (это долго по времени), то все папки с картинками и csv файл с разметкой должны быть в основной папке. После этого, для запуска тренировки модели нужно ввести следующую команду

```bash

python main.py --train --epochs 15

```

### Для точности можно повысить --epochs (чем выше - тем лучше). Число показывает сколько раз нейросеть прогонит через себя датасет, с каждым разом Loss будет уменьшаться.

### Сам датасет был взят по ссылке: https://www.kaggle.com/datasets/balabaskar/wonders-of-the-world-image-classification/data
