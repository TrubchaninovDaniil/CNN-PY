### В общем это уже код с весами, если есть картинка и нужно проверить, то введите команду

```bash

python main.py --predict test.jpg

```

### Если хочеться самому проверить и потренировать модель (это долго), то все папки с картинками и csv файл с разметкой должны быть в основной папке. После для запуска тренировки модели вводите волшебную команду

```bash

python main.py --train --epochs 15

```

### Для точности можно повысить --epochs (чем выше - тем лучше). Число показывает сколько нейронка прогонит раз через себя датасет
