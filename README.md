### В общем это уже код с весами, если есть картинка и нужно проверить 

```bash

python main.py --predict test.jpg

```

### Если хочеться самому проверить и потренировать модель (это долго), то 
### все папки с картинками дожны быть в основной папке и csv файл с разметкой
### после волшебная команда

```bash

python main.py --train --epochs 15

```

### для точности можно повысить --epochs (чем выше - тем лучше) 
### число показывает сколько нейронка прогонит через себя датасет
